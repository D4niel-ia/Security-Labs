# Dashboard de Inteligência de Ameaças e Indicadores de SI (Power BI)

## Descrição do Lab
Utilização de Power BI para a criação de dashboards executivos e operacionais baseados em logs de segurança. O projeto transforma dados brutos de firewalls e sistemas de detecção em inteligência acionável.

## Indicadores (KPIs) Monitorados
* **Volume de Incidentes por Severidade:** Classificação de eventos críticos vs. informativos.
* **Top Attack Vectors:** Identificação das origens mais frequentes de tráfego malicioso.
* **Eficiência de Bloqueio:** Comparação entre ameaças detectadas e ameaças mitigadas.

## Origem dos Dados
* Logs simulados de Honeypots e datasets públicos de tráfego de rede (PCAP analysis).
* Integração de dados de engajamento de usuários (adaptado para análise de comportamento de rede).

## Tecnologias Utilizadas
* Power BI Desktop / DAX
* ETL (Power Query) para limpeza de logs brutos.
* Markdown para documentação de métricas.

## Impacto no Negócio
* Facilitação da tomada de decisão para investimentos em infraestrutura de segurança.
* Comunicação clara de riscos para stakeholders não técnicos.
